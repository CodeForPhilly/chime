Introduction

CHIME Documentation

This book is the living documentation for developers and operators working on the CHIME project or running instances of it.

It is maintained in the GitBook format within the docs/ directory of the project's main Git repository, and new versions are published automatically upon merge to the master branch to: codeforphilly.github.io/chime

To contribute to this book, please open a pull request against the master branch of github.com/CodeForPhilly/chime.
